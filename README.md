# SOWWISE-INTEGRATED-FARMING-VILLAGE-BOY-FARMS-
The 500-acre Integrated Farming System in Oyo State, Nigeria, which has a budget of 2 billion NGN, integrates crops and animals.  To ensure long-term success, the project prioritizes sustainability, risk management, and financial viability, as well as frequent plan reviews.
